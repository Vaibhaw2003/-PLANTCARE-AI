🌿 About PlantCare AI

AI Powered Plant Disease Detection System

📌 Project Overview

PlantCare AI is a Machine Learning based web application designed to detect plant diseases from leaf images using advanced Transfer Learning techniques. The system helps farmers and agriculture professionals identify plant diseases early and take necessary preventive actions.

🎯 Our Objective

Our goal is to provide a fast, accurate, and user-friendly AI solution that can analyze leaf images and detect diseases in real-time. This reduces crop loss and improves agricultural productivity.

🧠 Technology Used

  • ✔ Python & Flask (Backend)
  • ✔ TensorFlow / Keras (Deep Learning Model)
  • ✔ Transfer Learning (Pre-trained CNN Models)
  • ✔ HTML, CSS & JavaScript (Frontend)

🚀 Key Features

  • ✔ Leaf Image Upload (Drag & Drop)
  • ✔ Real-Time Disease Prediction
  • ✔ Confidence Score with Progress Bar
  • ✔ Modern & Responsive UI

🌍 Future Scope

In future, PlantCare AI can be extended to support multiple crop types, mobile app integration, cloud deployment, and automated treatment suggestions.
